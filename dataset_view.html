Dataset View

class classes.class_dataset_view.DatasetView(name: str, **options)

Represents a view of a dataset.

name

The name of the dataset view. It must be a valid SQL name with lowercase letters.

Type:

str

description

A description of the dataset view. (Optional)

Type:

str

definition_query

The SQL query defining the dataset view. (Optional)

Type:

str

columns

List of columns in the dataset view. (Optional)

Type:

list of DatasetColumn

marked_for_deletion

Indicates whether the dataset view is marked for deletion. (Optional)

Type:

bool

Initialize a DatasetView object.

Parameters:

  • name (str) – The name of the dataset view. It must be a valid SQL name with lowercase letters.

  • **options – Additional options to specify description, definition_query, columns, and marked_for_deletion.

Raises:

ValueError – If the provided name violates the specified constraint or if columns is not a list of DatasetColumn objects.

Usage Example

Here’s an example of how to create and use a DatasetView object:

from starburst_api.classes import DatasetView, DatasetColumn

try:
    # Define some columns for the dataset view
    columns = [
        DatasetColumn(name="id", type="integer", description="Unique identifier."),
        DatasetColumn(name="name", type="string", description="Name of the item."),
        DatasetColumn(name="created_at", type="timestamp", description="Creation timestamp."),
    ]

    # Create a DatasetView instance
    view = DatasetView(
        name="customer_view",
        description="A view of customer data.",
        definition_query="SELECT id, name, created_at FROM customers",
        columns=columns,
        marked_for_deletion=False,
    )

    # Access attributes
    print(view.name)         # Output: "customer_view"
    print(view.description)  # Output: "A view of customer data."
    print(view.columns[0].name)  # Output: "id"

except ValueError as e:
    print(f"Error creating DatasetView: {e}")

# Example with invalid input
try:
    invalid_view = DatasetView(
        name="CustomerView",  # Invalid: contains uppercase letters
        columns=["invalid_column"]  # Invalid: not a list of DatasetColumn objects
    )
except ValueError as e:
    print(f"Error: {e}")  # Output: Error message about invalid input
