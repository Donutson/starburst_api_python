Relevant Link

class classes.class_relevant_link.RelevantLink(label: str, url: str)

Represents a relevant link.

label

The label of the relevant link.

Type:

str

url

The URL of the relevant link.

Type:

str

Initialize a RelevantLink object.

Parameters:

  • label (str) – The label of the relevant link.

  • url (str) – The URL of the relevant link.

Raises:

ValueError – If label or url exceeds the maximum length.

Usage Example

Here’s how to create and use a RelevantLink object:

from starburst_api.classes import RelevantLink

try:
    # Create a RelevantLink instance
    link = RelevantLink(
        label="Starburst Documentation",
        url="https://docs.starburst.io/latest/"
    )

    # Access attributes
    print(link.label)  # Output: "Starburst Documentation"
    print(link.url)  # Output: "https://docs.starburst.io/latest/"

except ValueError as e:
    print(f"Error creating relevant link: {e}")
